Projects - Back End
Firewall Design
Implement a basic firewall running at end host with real packets in a Linux-based virtual machine. Dealing with a real TCP/IP packet. Furthermore, the firewall is improved by implementing a stateful active application-layer firewall. Also dealing with HTTP and DNS protocols.
Processor Design
Implement a 2-stage pipelined processor for the Ida assembly language - similar to MIPS assembly language and use logisim as the primary software. The processor has 16 registers with each has 24 bits wide and each of them has its own convention. NOTE: image is blurred on purposed - to prevent any copy as it is part of class project's.
2048 Game with Expectimax AI
I developed my own representation of the 2048 game with Java. It adapts the original gameplay that with a ratio 9 to 10 it will generate a random 2/4 number in the current board. Further addition to the game is I developed my own AI with expectimax algorithm with my own heuristic function. After running a test of 1000 game, it achieves 60% getting a score of 1024. Currently I have several more optimization where I have not implement yet.
Twitter Trends
Develop a geographic visualization of Twitter data across the USA. The output map referenced from public tweets of a specific trend that have been tagged with geographic locations. Each tweet is assigned to a sentiment based on all the words it contains. Aggregate tweets by the state with the closest geographic center and finally give a color representation to each state to the aggregate sentiment of its tweets.
